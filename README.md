# 恐竜3Dバースデーカード

ブラウザだけで動く、インタラクティブな3D恐竜誕生日カードです。
Three.jsで描画した恐竜たちが画面内に登場し、ボタン操作や画面ドラッグで楽しめます。

## ファイル

- `index.html` — 最新版のカード本体。ブラウザで直接開くだけで動作します。

## 使い方

1. `index.html` をブラウザ(Chrome等)で開く
2. 「カードをあける！」ボタンを押してスタート
3. 画面をドラッグ/スワイプして3Dの世界を回転
4. 「ほえる！」「エサをあげる」「大ジャンプ！」ボタンで恐竜を操作

## 使用技術

- [Three.js](https://threejs.org/) — 3Dグラフィックス描画
- [Tailwind CSS](https://tailwindcss.com/) — UIスタイリング
- Web Audio API — 効果音(外部音声ファイル不要)
